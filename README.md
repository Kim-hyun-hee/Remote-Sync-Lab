# Remote-Sync-Lab
> 본 프로젝트는 프로덕션 배포용이 아닌 학습용 코드입니다.
> Photon Fusion 환경에서 네트워크 동기화를 구현하는 과정에서 발생하는 이슈를 직접 겪고 개선해 나가는 데 초점을 맞췄습니다.

1. Stroke 분리: 두 명이 동시에 선을 그렸을 때 서로 섞이지 않는지
3. Late Join 동기화: 1명이 먼저 그림을 긜고, 2번째 사용자가 늦게 참가했을 때 기존 선이 보이는지
4. Payload Chunking / Batching: 빠르게 드로잉하여 포인트가 많을 때도 끊김/오류 없이 동작하는지
